The first classes you should know about:

  • A 2D XYC image : IcyBufferedImage
  • The ZT manager of the images : Sequence
  • The abstract class to write plugins : Plugin_ / PluginActionable
  • The overlays to interact and draw rich information on image : Overlay (previously AbstractPainter / Painter)
